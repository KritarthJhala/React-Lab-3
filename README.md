This is a React-based web application that allows users to manage and monitor student performance in an interactive and organized way. The application displays a table of student names, scores, and pass/fail status based on predefined criteria. Users can add new students through a form and update existing scores dynamically, with changes reflected instantly on the page. A summary panel shows key statistics such as the total number of students, number of students who passed, and the average score, helping provide a quick overview of class performance.
